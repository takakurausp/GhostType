# GhostType

画面の文脈（コンテキスト）を理解し、最適なフォーマットでテキストを入力してくれるWindows向け音声入力アシスタントです。Google Gemini APIのマルチモーダル機能（音声＋画像）を活用し、あなたの思考をシームレスにテキスト化します。

## 🌟 主な機能

* **文脈を読んだスマートなテキスト化:** アクティブウィンドウのスクリーンショットをAIが読み取り、既存の文章の文体（だ・である、です・ます）やフォーマット（箇条書きなど）に合わせて自然な日本語を出力します。
* **ワンキーで完結するシームレスな操作:** `Ctrl + Space` を押して話し、もう一度押すだけで、現在カーソルがある場所にテキストが自動でペーストされます。
* **ハイブリッドなモード切り替え:** 話し出しの「合言葉」による自動判別に加え、`Shift + Space` キー操作による確実な「強制モード（メール、箇条書き等）」の切り替えに対応しています。
* **プロンプトのホットリロード機能:** AIへの指示（プロンプト）を外部テキストファイル（`prompt.txt`）から都度読み込むため、**アプリを起動したままプロンプトを書き換えて即座にテスト**することができます。
* **ケバ取り・自動整形:** 「えーっと」「あの」などの言い淀みを自動で削除し、適切な位置に句読点と改行を補完します。
* **スマートなポップアップUI:** 録音中や処理中のステータス、モード切り替えの確認表示が、画面中央下部（タスクバーのすぐ上）に現れ、用が済むと自動でスッと消えるため作業の邪魔になりません。

## 📦 必須環境

* **OS:** Windows 10 / 11 (Windows APIを利用しているため)
* **Python:** 3.9 以上
* **APIキー:** Google Gemini APIキー (Google AI Studioにて無料で取得可能)

## 🛠 インストール方法

1. リポジトリをクローンまたはダウンロードします。
2. コマンドプロンプトまたはPowerShellを開き、必要なライブラリを一括インストールします。（※最新の `google-genai` SDKを使用しています）

```bash
python -m pip install google-genai pyaudio pyautogui pyperclip pillow

```

> **※注意 (PyAudioのインストールエラーについて)**
> Windows環境で `pyaudio` のインストールに失敗する（赤文字のエラーが出る）場合は、以下のコマンドを順番に実行してインストールしてください。
> ```bash
> python -m pip install pipwin

> ```

## ⚙️ 初期設定

本アプリはセキュリティと利便性を考慮し、設定値やプロンプトをPythonコードから分離しています。以下の2つのファイルを設定してください。

### 1. 起動用バッチファイル（`start_ghosttype.bat`）の設定

同梱のバッチファイルをテキストエディタで開き、取得したAPIキーを入力します。また、用途に合わせてAIモデルやスクショ送信の有無を変更できます。

```bat
set GEMINI_API_KEY=あなたの_API_キーを_ここに_入力
set GEMINI_MODEL_NAME=gemini-3.1-flash-lite-preview
set GHOSTTYPE_SEND_SCREENSHOT=true

```

* **モデルの指定 (`GEMINI_MODEL_NAME`)**
* 高精度・複雑な処理向け: `gemini-3.1-flash-lite-preview` （デフォルト）
* 更に高精度・複雑な処理向け（高コスト）: `gemini-3-flash-preview`
* 互換性重視（高速・低精度・低コスト）: `gemini-2.5-flash`


* **スクリーンショットの送信 (`GHOSTTYPE_SEND_SCREENSHOT`)**
* `true`: 画面の文脈を読み取ります（デフォルト）。
* `false`: 画面を読み取りませんが、通信データ量が減り処理が高速化します。



### 2. プロンプトファイル（`prompt.txt`）の準備

AIの振る舞いを決める指示書です。同梱の `prompt.txt` をそのまま利用するか、ご自身の好みに合わせてルールを書き換えてください。
（※文字化けを防ぐため、必ず **UTF-8** 形式で保存してください）

## 🚀 使い方

1. `start_ghosttype.bat` を実行します。（※一部のアプリ上でホットキーが効かない場合には、バッチファイルを右クリックして「管理者として実行」してください）
2. 待機状態になったら、文字を入力したいアプリ（メモ帳、ブラウザ、メールソフトなど）を開き、カーソルを合わせます。
3. **`Ctrl + Space`** を押します。画面下部に「🎙️ 録音中...」と表示されたらマイクに向かって話します。
4. 話し終えたら再度 **`Ctrl + Space`** を押します。「⏳ 処理中...」のアニメーションのあと、自動的にテキストが貼り付けられます。

### 🗣️ モード切り替え（手動と自動）

GhostTypeは、キーボードによる「確実な手動切り替え」と、音声の合言葉による「スピーディな自動判別」の両方に対応しています。

#### 【確実な切り替え：強制モード】

待機中に **`Shift + Space`** を押すたびに、「🤖 自動(合言葉) ➔ ✉️ 強制: メール ➔ 📝 強制: 箇条書き ➔ ❓ 強制: 質問回答」の順でモードが切り替わります（画面下部に一瞬表示されます）。
強制モードを選択した状態で録音を開始すると、合言葉を言わなくても確実にそのモードのフォーマットで出力されます。

#### 【スピーディな切り替え：マジックワード（自動モード時）】

「🤖 自動(合言葉)」モードの状態で録音を開始し、以下の「合言葉」から話し始めることで特別なモードが発動します。（出力されるテキストからは合言葉自体は自動で削除されます）

* **「メール、[内容]」**
* 画面から宛名を推測し、ビジネスメールの挨拶と結びの言葉を自動で補完して出力します。


* **「箇条書き、[内容]」**
* 話した内容を要約し、見やすい箇条書きのフォーマットに変換して出力します。


* **「AIに質問、[質問内容]」**
* 画面の文脈を無視し、純粋な質問への「回答のみ」をポンッと出力します。（例：「AIに質問、日本の首都は？」→「東京」）


* **（合言葉なし）**
* 通常モード。画面の文脈を読み取り、既存の文章に合わせて適切な改行や句読点を補ってテキスト化します。



## 🔧 プロンプトのカスタマイズ（ホットリロード）

GhostTypeの挙動（メールの署名や、独自のモード追加など）を変えたい場合は、**アプリを起動したまま** `prompt.txt` を編集・上書き保存してください。次の録音からすぐに新しいルールが適用されます。

## ⚠️ 注意事項・制限事項

* 本ツールはWindows OSのネイティブAPI（Win32 API）を使用しているため、MacやLinuxでは動作しません。
* 処理中（アニメーション表示中）に他のウィンドウへフォーカスを移すと、誤った場所にペーストされる可能性があります。処理が終わるまでそのままお待ちください。
* クリップボードの履歴（Win+V）に、直近に生成されたテキストが残ります。
* Gemini APIの利用にはデータ量に応じた料金が発生する場合があります。利用状況はご自身で管理してください。
