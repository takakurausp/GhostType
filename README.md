# GhostType

画面のスクリーンショットやクリップボードのテキストなど、PC上の「文脈（コンテキスト）」を理解し、最適なフォーマットでテキストを入力・生成してくれるWindows向け音声入力アシスタントです。Google Gemini APIを活用し、あなたの思考をシームレスにテキスト化します。

## 🌟 主な機能

* **選べる8つの強力なモード:** 単なる文字起こしから、クリップボードの内容を踏まえたメール返信、画面を見ながらのAIへの質問まで、用途に合わせた8つの独立モードを搭載。
* **ワンキーで完結するシームレスな操作:** `Ctrl + Space` を押して話し、もう一度押すだけで、現在カーソルがある場所にテキストが自動でペーストされます。
* **誤爆を防ぐスマートなモード切替:** `Win + Alt + Space` を1回押すと「現在のモード確認」、そのまま2秒以内にもう1回押すと「モードの切り替え」が行われる、確実で安全なUXを採用。
* **短時間キャンセル機能:** 誤って録音キーに触れてしまった場合（録音時間が1秒未満の場合）は、無駄なAPI通信を行わず即座にキャンセルされます。
* **プロンプトのホットリロード機能:** AIへの全体的な指示（フィラー除去のルールや文体の指定など）を外部の `prompt.txt` から読み込むため、アプリを起動したままルールを追加・変更できます。
* **スマートなポップアップUI:** ステータスやモードの確認表示は、画面中央下部（タスクバーのすぐ上）に現れ、用が済むと自動でスッと消えるため作業の邪魔になりません。

## 📦 必須環境

* **OS:** Windows 10 / 11 (Windows APIを利用しているため)
* **Python:** 3.9 以上
* **APIキー:** Google Gemini APIキー (Google AI Studioにて無料で取得可能)

## 🛠 インストール方法

1. リポジトリをクローンまたはダウンロードします。
2. コマンドプロンプトまたはPowerShellを開き、必要なライブラリを一括インストールします。

```bash
python -m pip install google-genai pyaudio pyautogui pyperclip pillow

```

> **※注意 (PyAudioのインストールエラーについて)**
> Windows環境で `pyaudio` のインストールに失敗する（赤文字のエラーが出る）場合は、以下のコマンドを順番に実行してインストールしてください。
> ```bash
> python -m pip install pipwin
> pipwin install pyaudio
> 
> ```

## ⚙️ 初期設定

本アプリを安全に利用するため、APIキーやプロンプトなどはコードから分離されています。

### 1. APIキーの設定（推奨：ユーザー環境変数）

セキュリティの観点から、APIキーはWindowsの「ユーザー環境変数」に登録することを強く推奨します。

1. Windowsのスタートメニューを開き、「**環境変数**」と検索します。
2. 「システム環境変数の編集」または「アカウントの環境変数の編集」を開きます。
3. 「ユーザー環境変数」の「新規」ボタンを押し、以下のように入力して保存します。
* 変数名: `GEMINI_API_KEY`
* 変数値: `取得したAPIキー`


4. 設定を反映させるため、開いているコマンドプロンプトやターミナルがあれば一度すべて閉じます。

> **⚠️ 警告: バッチファイルへの直接書き込みについて**
> 起動用の `start_ghosttype.bat` 内に `set GEMINI_API_KEY=...` と直接キーを書き込んで動かすことも可能ですが、ファイルにパスワードが丸見え（平文）で保存される状態になります。この方法は、**「絶対に自分以外の人間が触らない、完全にプライベートなPC」でのみ**利用してください。

### 2. 起動用バッチファイル（`start_ghosttype.bat`）の設定

同梱のバッチファイルでは、用途に合わせてAIモデルを変更できます。

```bat
set GEMINI_MODEL_NAME=gemini-3.1-flash-lite-preview

```

* **モデルの指定 (`GEMINI_MODEL_NAME`)**
* 高精度・複雑な処理向け: `gemini-3.1-flash-lite-preview` （デフォルト）
* 更に高精度・複雑な処理向け（高コスト）: `gemini-3-flash-preview`
* 互換性重視（高速・低精度・低コスト）: `gemini-2.5-flash`



### 3. プロンプトファイル（`prompt.txt`）の準備

AI全体に適用させたい「基本ルール」を記述します。同梱の `prompt.txt` をそのまま利用するか、ご自身の好みに合わせて書き換えてください。（※必ず **UTF-8** 形式で保存してください）

**【おすすめの設定例】**

```text
・音声認識時の「えー」「あの」「その」などのフィラー（ケバ）は完全に削除してください。
・ユーザーが途中で言い間違えて言い直している（自己訂正している）場合は、言い間違えた部分を削除し、直後の正しい言葉を優先して意味が通るように整えてください。
・文末は、特に指定がない限り「です・ます調」の丁寧語に統一してください。

```

## 🚀 使い方

1. `start_ghosttype.bat` を実行します。（※一部のアプリ上でホットキーが効かない場合には、バッチファイルを右クリックして「管理者として実行」してください）
2. 待機状態になったら、文字を入力したいアプリ（メモ帳、ブラウザ、メールソフトなど）を開き、カーソルを合わせます。

### 🎙️ 録音の操作 (`Ctrl + Space`)

* **`Ctrl + Space`** を押すと録音が開始されます（画面に赤いマイクアイコンが表示されます）。
* 話し終えたら再度 **`Ctrl + Space`** を押します。「⏳ 処理中...」のアニメーションのあと、自動的にテキストが貼り付けられます。
* ※誤って押してしまった場合でも、1秒以内に再度押せばキャンセル扱いとなります。

### 🔄 モードの確認と切り替え (`Win + Alt + Space`)

* **1回押す:** 画面下部に👀アイコンと共に「現在のモード」が2秒間表示されます。（確認のみ）
* **表示中にもう1回押す:** ➔アイコンに変わり、次のモードへ切り替わります。連打することで素早く目的のモードに合わせることができます。

## 🎯 搭載モード一覧

GhostTypeは、用途と参照させたい情報（クリップボードやスクショ）に合わせて、以下の8つのモードを搭載しています。

1. **🗣️ 口述のみ**: 純粋な文字起こしと文章整形を行います。（参照情報なし）
2. **📋 口述+クリップ**: クリップボード内のテキストを専門用語などの「参照情報」として活用し、文字起こしの精度を高めます。
3. **🖼️ 口述+スクショ**: 現在の画面スクショを「参照情報」として活用し、文字起こしの文脈精度を高めます。
4. **✉️ メール**: 音声内容をもとに、ビジネスメールの形式（挨拶、結びなど）を自動補完して作成します。
5. **📧 メール+クリップ**: クリップボード内のテキストを「受信メール」とみなし、それに対する返信メールを作成します。
6. **🤖 AI**: 音声を「AIへの質問・指示」として扱い、回答のみを出力します。
7. **🤖📋 AI+クリップ**: クリップボード内のテキストについて、AIに質問・指示を出します。（例：「これを要約して」等）
8. **🤖🖼️ AI+スクショ**: 画面スクショについて、AIに質問・指示を出します。（例：「このエラーの意味を教えて」等）

## ⚠️ 注意事項・制限事項

* 本ツールはWindows OSのネイティブAPI（Win32 API）を使用しているため、MacやLinuxでは動作しません。
* 処理中（アニメーション表示中）に他のウィンドウへフォーカスを移すと、誤った場所にペーストされる可能性があります。処理が終わるまでそのままお待ちください。
* クリップボードの履歴（Win+V）に、直近に生成されたテキストが残ります。
* Gemini APIの利用にはデータ量に応じた料金が発生する場合があります。画像（スクショ）を送信するモードはテキストのみのモードより消費量が多いため、利用状況はご自身で管理してください。
